# 🔐 Password Strength Analyzer

A simple and interactive web-based Password Strength Analyzer built using **HTML, CSS, and JavaScript**.  
This project helps users check the strength of their passwords in real time and encourages the use of strong and secure passwords.

## 📌 Features

- 🔎 Real-time password strength checking  
- 👁️ Show/Hide password toggle  
- 📊 Visual strength indicator bar  
- ⚡ Instant feedback (Weak / Medium / Strong)  
- 💡 Security tips for better passwords  
- 📱 Responsive and modern UI design

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
